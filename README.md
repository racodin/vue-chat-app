# vue-chat-app

## 구현목록

### 입력폼

- 빈 문자열 입력 체크
- 입력 문자열 길이 체크
- 입력 문자 엔터키 전송
- 입력 문자와 이전 문자의 중복 체크
- 짧은 시간안에 연속적인 입력 방지
- 욕설 체크 및 별표 치환

### 채팅 리스트

- 리스트 데이터 추가
- 리스트 추가시 스크롤 하단 정렬
- 리스트 최대 길이 제한 및 제거

### 이모티콘

- 유니코드 이모티콘
- 이모티콘 선택 입력
